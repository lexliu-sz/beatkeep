# Beatkeep

A tap-tempo meter that grades how consistent your taps are. iPhone & iPad, free, no ads, no in-app purchases.

## How to use

1. Tap anywhere on the screen at a steady pace.
2. After **2 taps**, your live BPM appears.
3. After **3 taps**, a stability score (0–100%) appears.
4. Pause for 3 seconds; the next tap starts a fresh measurement.
